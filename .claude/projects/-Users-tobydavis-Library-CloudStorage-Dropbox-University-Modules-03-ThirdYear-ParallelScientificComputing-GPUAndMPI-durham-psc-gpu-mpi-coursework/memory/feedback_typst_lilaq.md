When generating plots for the report, use Lilaq (a Typst plotting library) directly in the Typst document rather than Python/matplotlib. Data processing can be done in Typst itself.

**Why:** The report is written in Typst and the user wants consistent, integrated plots.

**How to apply:** When asked to create graphs/plots for the report, write Typst code using Lilaq. Don't generate Python plotting scripts.
